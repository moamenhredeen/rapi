# Screens Overview

rapi has three main screens, accessed via the activity bar on the left:

| Icon | Screen | Description |
|------|--------|-------------|
| 📦 | Collections | Send and test HTTP requests |
| 🌍 | Environments | Manage environments and variables |
| ⚙ | Settings | App preferences (theme, etc.) |

The app window is divided into three areas:

- **Activity Bar** — A narrow icon bar on the far left for switching between screens.
- **Content Area** — The main area that renders the active screen.
- **Status Bar** — A thin bar at the bottom showing status messages.
