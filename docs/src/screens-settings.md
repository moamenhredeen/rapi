# Settings Screen

The Settings screen provides application preferences that persist across sessions.

## Theme

Choose from 22 built-in themes via the dropdown:

- Light, Dark
- Dracula, Nord
- Solarized Light, Solarized Dark
- Gruvbox Light, Gruvbox Dark
- Catppuccin Latte, Frappé, Macchiato, Mocha
- Tokyo Night, Tokyo Night Storm, Tokyo Night Light
- Kanagawa Wave, Dragon, Lotus
- Moonfly, Nightfly
- Oxocarbon, Ferra

The selected theme is applied instantly to the entire application and saved to the configuration file.
